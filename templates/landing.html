Fleet Intelligence

Mobile-First GPS Fleet Tracking

Municipal vehicles tracked in real time via mobile GPS without dedicated hardware. Records location logs, speed, and status for every vehicle in the Harare City Council fleet.

  • Live vehicle position updates
  • Driver assignment & trip history
  • Active / Idle / Offline monitoring
  • Speed & utilisation analytics
Route Engine

Dynamic Route Optimisation

Dijkstra-based shortest path algorithms dynamically recalculate routes using live congestion multipliers. Traffic events trigger faster alternative routes in real time.

  • Graph-based node/edge road network
  • Congestion-weighted travel times
  • Precomputed route storage
  • Reduces idle time by up to 40%
Traffic Awareness

Live Traffic Event Monitoring

Traffic incidents are logged with GPS coordinates and severity scores. WebSocket push notifications propagate events to affected routes within seconds.

  • Low / Medium / High severity events
  • Geo-tagged incident reporting
  • Congestion multiplier propagation
  • Operator alert broadcasts
System Evaluation

Performance Metrics Dashboard

Dedicated metrics module measuring average travel time, route efficiency, fleet utilisation rates, and service request response times over time.

  • Avg travel & response time tracking
  • Route efficiency score (0–100)
  • Fleet utilisation % per day
  • Historical trend comparison
