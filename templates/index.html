Fleet Intelligence

Mobile-First GPS Fleet Tracking

Municipal vehicles are tracked in real-time via mobile GPS without requiring dedicated hardware installations. The system records location logs, speed, and status for every vehicle in the Harare City Council fleet.

  • ✓Live vehicle position updates
  • ✓Driver assignment & trip history
  • ✓Active / Idle / Offline status monitoring
  • ✓Speed & utilization analytics
Route Engine

Dynamic Route Optimization

Dijkstra-based shortest path algorithms dynamically recalculate routes using live congestion multipliers. When a traffic event is reported, the system automatically suggests faster alternatives in real time.

  • ✓Graph-based node/edge road network
  • ✓Congestion-weighted travel times
  • ✓Precomputed route storage
  • ✓Reduces idle time by up to 40%
Traffic Awareness

Live Traffic Event Monitoring

Traffic incidents — accidents, roadworks, congestion — are logged with GPS coordinates and severity scores. The system propagates these events to affected routes within seconds using WebSocket push notifications.

  • ✓Low / Medium / High severity events
  • ✓Geo-tagged incident reporting
  • ✓Congestion multiplier propagation
  • ✓Operator alert broadcasts
System Evaluation

Performance Metrics Dashboard

Objective 5 of the research is addressed through a dedicated metrics module that measures average travel time, route efficiency scores, fleet utilization rates, and response time to service requests over time.

  • ✓Avg travel & response time tracking
  • ✓Route efficiency score (0–100)
  • ✓Fleet utilization % per day
  • ✓Historical trend comparison
Research Context
Why Harare Needs This System
🏙️
Urban Logistics Crisis
Delivery vehicles in congested urban environments spend 28–40% of their time idling, generating economic losses and environmental degradation in cities like Harare.
Kai et al., 2025
📡
Software-First Approach
Wagga Wagga City Council saved $1.1M through data-driven fleet management. This system replicates that model at zero hardware cost using mobile GPS and open-source Django infrastructure.
IPWEA, 2025
🌍
African Innovation Model
Nairobi logistics companies achieved major efficiency gains using AI-powered route optimization on basic smartphones — proving that frugal engineering can transform Sub-Saharan urban logistics.
BusinessDay Nigeria, 2025
🏛️
Ward 14 Case Study
Kambuzuma residents face absent refuse collection, deteriorated roads, and failed water infrastructure — all symptoms of broken logistics governance that this system directly targets.
Mutandiro, 2025
🔗
Digital Twin Precedent
Chattanooga, Tennessee's multi-domain urban mobility platform demonstrates the feasibility of software-centric traffic management replicable in resource-constrained municipalities.
Xu et al., 2023
📈
Scalable to Zimbabwe
Designed to scale across Zimbabwean municipalities with variable internet connectivity. All core tracking, routing, and alerting functions operate with intermittent data availability.
Razemba, 2026
